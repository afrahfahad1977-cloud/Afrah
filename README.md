# منصة كلية العلوم

منصة ويب تعريفية مبسطة لكلية العلوم تعرض التخصصات ونبذة مختصرة عن كل تخصص ومجالاته.

## المميزات
- عرض التخصصات العلمية في بطاقات منظمة.
- نبذة عن كل تخصص مع أبرز مجالاته.
- **صفحة تفصيلية لكل تخصص** (نبذة موسّعة، المقررات، ومجالات العمل).
- قسم **الرؤية والرسالة والقيم وكلمة العميد**.
- قسم **القبول والتسجيل** وشروطه.
- قسم **اتصل بنا** ببيانات التواصل.
- خاصية البحث الفوري عن التخصصات.
- تصميم متجاوب يدعم اللغة العربية (RTL).

## التشغيل
افتح ملف `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا:

```bash
python3 -m http.server 8000
```

ثم افتح: <http://localhost:8000>

## الملفات
- `index.html` — الصفحة الرئيسية (التخصصات، الرؤية، القبول، الاتصال).
- `major.html` — صفحة تفاصيل التخصص.
- `styles.css` — التنسيقات.
- `data.js` — بيانات التخصصات المشتركة.
- `app.js` — منطق الصفحة الرئيسية والبحث.
- `major.js` — منطق صفحة التفاصيل.

## إضافة تخصص
عدّل مصفوفة `MAJORS` داخل `data.js` بإضافة عنصر جديد يحتوي على معرّف فريد (`id`)،
والأيقونة، والاسم، والنبذة، والمجالات، والمقررات، ومجالات العمل.
